## java课程设计: 通讯录管理系统

- Spring SpringMVC Hibernate JQuery 
- 支持联系人增删查改
- 支持分组增删
- 支持根据号码,姓名搜索联系人
- 支持联系人分组移入移出
- 导入导出等功能可自行添加
